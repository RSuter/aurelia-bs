# bs-tabs

```html
<bs-tabs>
    <bs-tab header="Tab 1">
        My Content
    </bs-tab>
    <bs-tab header="Tab 2">
        My Content
    </bs-tab>
</bs-tabs>
```

## Bindable properties

- selectedTabId

## Bindable properties of bs-tab

- id
- header
- panel: Specifies whether to render a panel
- visible
- style

# bs-textbox

```html
<bs-textbox label="My Label"
            value.bind="myText & bsValidate">
</bs-textbox>
```

## Bindable properties

- label
- value (two-way)
- placeholder
- enabled
- readonly
- help
- rows
- style

## Methods

- **focus():** Focuses the textbox
- **selectAll():** Selects all texts in the textbox

# bs-select

```html
<bs-select label="My Label"
           value.bind="myValue & bsValidate"
           items.bind="myItems">
</bs-select>
```

## Bindable properties

- label
- items
- value (two-way)
- values (two-way)
- displayPath
- valuePath
- required
- enabled
- readonly
- help
- multiple
- height
